Simple Pace Calculator.

Add pace in Minutes per KM and distance in KM.

Press submit to see the time it would take to complete that distance at that pace.

# Dev notes

Run `pyuic4 pacecalculator-qt.ui -o pacecalculatorqtui.py` to generate Python class from QT Designer file.
Run `pyrcc4 pacecalculator.qrc -o pacecalculator_rc.py` to generate Python class from QT Resources file.
